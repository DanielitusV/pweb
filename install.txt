# Instrucciones de instalación - Servidor INTERAQUIZ (Windows 11)

Este archivo contiene las instrucciones para configurar una PC como 
servidor del proyecto INTERAQUIZ.

El frontend es HTML/CSS/JS puro, pero también se necesita exponer el
contenido a Internet y trabajar con base de datos (MongoDB)


## Requisitos obligadorios para el servidor (PC Windows)

1. **Visual Studio Code**
    - Descargar desde: https//code.visualstudio.com/

2. **Extensión Live Server para VS Code**
    - Abre VS Code -> Extensiones -> Buscar "Live Server" -> Instalar
    - Con este componente se permite levantar el HTML en `localhost:5500`

3. **Ngrok**
    - Descargar desde: https://ngrok.com/download
    - Crear cuenta gratuita
    - Autenticarse con token:
    ```bash
    ngrok config add-authtoken <TOKEN-PROPIO>
    ```

4. **Git (opcional para clonar repositorio)**
    - https://git-scm.com/download/win

***

## Como levanta el proyecto localmente

1. Abre el proyecto en Visual Studio Code
2. Abre `index.html` y haz clic derecho -> **"Open with Live Server"**
3. El sitio estará corriendo en:
    `http://localhost:5500`

***

## Cómo exponer el servidor por internet (Ngrok)

1. En tu terminal, escribe:
    ```bash
    ngrok http 5500
    ```
